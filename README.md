# Skipass

## Требования

PHP

https://windows.php.net/download/

Composer - менеджер зависимостей

https://getcomposer.org/download/

## Запуск

Клонировать проект к себе на компьютер

Установить зависимости

```
composer install # или composer.phar install если он установлен в виде файла .phar
```


Для запуска локального сервера из папки с проектом пишем

```
$ php -S localhost:8081
```

После этого в браузере открываем адрес http://localhost:8081
